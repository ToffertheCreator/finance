# 💸 Personal Finance Tracker (WIP)
> A simple, object-oriented personal finance tracker app built with Python and KivyMD. Track transactions, set budgets, view analytics, and manage your savings — all in one lightweight desktop GUI.

## 🚀 Features
🧾 Transaction Tracking – Add income or expenses with category tags and dates
💰 Savings Manager – Track your current savings and history
📊 Analytics – View graphs
🖼️ Modern UI – Built with KivyMD using Material Design components

## 🧱 Tech Stack
Python 3.10+
Kivy 2.3.1
KivyMD 1.2.0
